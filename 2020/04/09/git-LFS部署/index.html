一次获取LFS对象的最近版本

7 表示同时下载过去 7 天内的版本（相对于获取的 LFS 对象的时间），该项配置默认值为 0，即不获取过去的版本，而仅获取指定的版本。
有了对最近的定义后，我们可以选择在执行 git lfs fetch 命令时，加上 –recent 参数以同时获取最近版本；
从而总是同时获取 LFS 对象的最近版本。
